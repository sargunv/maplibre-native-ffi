This planned contributor note covers Java (FFM) binding design, implementation,
tests, and reference generation.

Tracking issue: [Add Java (FFM) bindings](https://github.com/sargunv/maplibre-native-ffi/issues/45).

Planned coverage:

- package and module layout;
- mapping C handles to Java FFM-backed types;
- ownership and lifetime conventions;
- status, error, and diagnostic mapping;
- event draining and callback safety;
- generated Java reference documentation;
- Java tests and examples.
